## Description

distance` is an integer specifying the minimum separation between peaks during peak finding. It is used in `ExtractAllRawPeaks`, `SmoothData_and_FindPeaks`, `ToolsGaussianChrom`, `ResolvingGaussianChromatogram`, `SplitParametersMat`, and `ResolveChromPeaks` to control peak detection and separation.
