## Description

LateLoc` is an integer representing the end boundary of a chromatographic peak, used by multiple functions (`IntegrateChromatographicPeak`, `BaseLine`, `ResolvingChromatogram`, `SummarizeChPeak`, `RT_Gauss_std`, and `Summarize_ms1_ChPeak`) to define the end of a peak for processing and analysis.
