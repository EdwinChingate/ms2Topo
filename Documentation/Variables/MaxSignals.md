## Description

MaxSignals` is an integer, used by `SmoothFourier`, `ToolsGaussianChrom`, `ResolvingGaussianChromatogram`, and `SmoothData_and_FindPeaks` to limit the maximum number of data points used when smoothing a chromatogram, in order to reduce computation time and ensure efficient data processing.
