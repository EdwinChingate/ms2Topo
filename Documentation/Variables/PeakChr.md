## Description

PeakChr` is a numpy array that holds data for a chromatographic peak, and it is used by `SmoothFourier`, `SmoothSavgol`, `RedistributeSampling`, and `SmoothPeak` to perform smoothing, filtering, and resampling of chromatographic data.
