## Description

int_col` is an integer specifying the column index that contains intensity values within a chromatogram or spectrum. It is used in `SmoothFourier`, `IntegrateChromatographicPeak`, `ToolsGaussianChrom`, `ResolvingGaussianChromatogram`, `SmoothSavgol`, `RedistributeSampling`, `BaseLine`, `UmbrellasStats`, `ResolvingChromatogram`, `Chrom_ms1Peaks_Summaries`, `RT_Gauss_std`, `RefineChromMat`, `Summarize_ms1_ChPeak`, `ReResolvingChromatograms`, `Feat_RT_edges`, `PlotChromatogram` and `SmoothPeak` to select the appropriate column for intensity-related operations and processing.
