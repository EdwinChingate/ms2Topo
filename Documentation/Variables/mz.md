## Description

mz` is a numerical value representing a mass-to-charge ratio, used across multiple functions such as `ms2Peak`, `ms2Peak_contribution`, `GaussianPeak`, `MaxIntChromatogram`, `Find_ms2Peak`, `Chrom_ms1Peaks_Summaries`, `closest_ms1_spec`, `ms2_peak_stats`, `mzPeak`, `Find_mzPeak`, `ResolveFullChromatogram`, `ResolveChromPeaks`, `RetrieveChromatogram`, `AllSubChromatograms`, `PlotChromatogram`, `ModuleChromSummaries`, and `ms2_peakStats_safe` as a central value for locating, characterizing, and processing peaks in mass spectrometry data.
