## Description

minPoly` is an integer specifying the minimum polynomial order for Savitzky-Golay smoothing, used in `IntegrateChromatographicPeak`, `SmoothSavgol`, `ResolvingChromatogram`, `SmoothData_and_FindPeaks`, `PlotChromatogram`, and `SmoothPeak` for signal processing.
