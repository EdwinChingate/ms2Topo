## Description

Tanimoto_tol` is a float defining the minimum Tanimoto coefficient for two spectra to be considered neighbors. It's used as a threshold in `CheckNeigbours`, `Similarity_AdjacencyMatrix`, and `SimilarityNeigbours` and `GoodNeigbour` to establish spectral similarity.
