## Description

SummMS2 is a NumPy array containing summarized MS2 spectra data (m/z, RT, spectrum IDs, etc.). It's used in several functions like `ms2_SpectralRedundancy` for spectral clustering, `Similarity_AdjacencyMatrix` and `SimilarityNeigbours` for similarity calculations, `closest_ms1_spec` for finding the closest MS1 spectra, `ms2_features_stats` for feature extraction, `feat_ms2_Gauss` for feature generation, and `Retrieve_csv_ms2Spectra` to load MS2 summaries.
