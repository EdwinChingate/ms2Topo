## Description

SummarizeChFeat` is a list of lists or a NumPy array that stores the summarized chromatographic features, used in `Chrom_ms1Peaks_Summaries` to store the result of the summarization of chromatographic peaks, and in `ModuleChromSummaries` to store the features of a chromatogram module.
