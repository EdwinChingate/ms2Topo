## Description

ParametersMat` is a numpy array of floats that is a matrix of parameters for multiple Gaussian peaks, where each row is a peak and each column is a parameter (RT, std, area), and it is used by `MutantOffspring`, `OverlappingGaussPeaks`, `SplitParametersMat`, `PopulatorFromVec`, `RawGaussCouple`, `RestrictStdPeaks`, `RemoveChromBackGround`, `RawParametersCut`, `RawGaussParameters`, `ParametersFitGaussPeaks`, `UniquePeaks`, `First_GaussPar_Population`, `RefineChromPeak`, `RefineChromMat`, `EvaluatingCombinationsFromVec`, `ParametersFitGaussParallelPeaks`, `RawGaussSeed`, and `RefineParameters` to manage peak parameters during fitting, refinement, and other operations.
