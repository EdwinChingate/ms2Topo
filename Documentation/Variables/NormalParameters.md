## Description

NormalParameters` is a list or a numpy array of floats that contains the parameters of a Gaussian fit to a peak, including mz, mz_std, I_total and r2, and is used by `Normal_Fit`, `ms2_features_stats`, and `ms2_peak_stats`, as an output when fitting a Gaussian curve to a peak in a mass spectrum.
