## Description

SamplesInfDF` is a Pandas DataFrame holding sample information, used across multiple functions (`Samples_AttributesFilter`, `SamplesNames_from_List`, `Samples_NFeatures_Filter`, `RemoveBlankFeatures`, `SamplesNames_common_Attributes`) to filter, retrieve, and process samples based on their attributes.
