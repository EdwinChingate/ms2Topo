## Description

alpha` is a float representing the significance level for calculating confidence intervals in `ms2_peak_stats`, `ms2_features_stats` and `ms2_peakStats_safe`. Although passed as an argument, it is not used within `Chrom_ms1Peaks_Summaries`, `RetrieveChromatogram`, `PlotChromatogram`, and `ModuleChromSummaries`.
