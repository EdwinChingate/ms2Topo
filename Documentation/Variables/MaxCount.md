## Description

MaxCount` is an integer defining the maximum number of attempts for recursive peak-finding, used by `ms2Peak`, `Find_ms2Peak`, `Chrom_ms1Peaks_Summaries`, `ms2_peak_stats`, `ms2_features_stats`, `mzPeak`, `Find_mzPeak`, `RetrieveChromatogram`, `PlotChromatogram`, `feat_ms2_Gauss`, `ModuleChromSummaries`, `ms2_peakStats_safe`, and `ms2_spectrum` to avoid infinite loops in peak-finding algorithms.
