## Description

edgesVec` is a list or NumPy array containing the start and end locations of a chromatographic or spectral peak. It is used in `PeakChromStats` to define the boundaries of a chromatographic peak for calculating statistics, and in `PeakFeatStats` to do the same for a spectral peak.
