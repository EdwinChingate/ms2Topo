## Description

min_Int_Frac` is a numerical value that defines the minimum intensity fraction for a peak to be considered valid during spectral comparisons and alignments, used in `ms2_SpectralRedundancy`, `CheckNeigbours`, `Similarity_AdjacencyMatrix`, `SimilarityNeigbours`, `GoodNeigbour`, `Similarity_2Spectra`, `ms2_FeaturesDifferences`, `Features_ms2_SamplesAligment`, `SpectralOrder`, `ms2_SpectralSimilarityClustering`, and `Retrieve_and_Join_ms2_for_feature` to filter out less significant data.
