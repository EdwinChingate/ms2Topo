## Description

ToAdd` is a string, used in `ms2_SpectralRedundancy`, `ms2_FeaturesDifferences`, `Features_ms2_SamplesAligment`, `ms2_SpectralSimilarityClustering`, and `Retrieve_and_Join_ms2_for_feature`, to add a suffix to sample names when creating file paths or filenames related to MS2 spectra data.
