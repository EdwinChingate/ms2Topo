## Description

max_RT` is a numerical value that defines the upper limit of the retention time used in `ms2_SpectralRedundancy`, `mz_RT_spectrum_filter`, `AllMS2Data`, `ExtractAllRawPeaks` and `MS_L_IDs`, for filtering and data extraction.
