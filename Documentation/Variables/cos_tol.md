## Description

cos_tol` is a float used as a threshold for cosine similarity. It defines the minimum cosine similarity value for considering two spectra similar in  `ms2_SpectralRedundancy`, `ms2_FeaturesDifferences`, `Features_ms2_SamplesAligment`, `ms2_SpectralSimilarityClustering`, `CheckNeigbours`, `Similarity_AdjacencyMatrix`, `SimilarityNeigbours` and `GoodNeigbour`.
