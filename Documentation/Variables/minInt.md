## Description

minInt` is a numerical value defining the minimum intensity threshold. It's used across multiple functions such as `All_ms2_spectra`, `ms2Peak`, `Find_ms2Peak`, `Chrom_ms1Peaks_Summaries`, `ms2_peak_stats`, `mzPeak`, `Find_mzPeak`, `SplitChromatogram`, `RetrieveChromatogram`, `PlotChromatogram`, `ModuleChromSummaries`, `ms2_peakStats_safe`, and `ms2_spectrum` to filter out low-intensity signals and peaks.
