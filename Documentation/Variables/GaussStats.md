## Description

GaussStats` is a list of floats or a numpy array containing parameters from a Gaussian fit, used to statistically describe peaks in mass spectrometry data such as their mean, standard deviation, and area, and is used by functions like `mz_Gauss_std`, `PeakChromGauss` and `RT_Gauss_std`.
