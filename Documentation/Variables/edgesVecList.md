## Description

edgesVecList` is a list of lists, where each inner list defines the start and end indices of a feature. It is used in `mz_Edges` to process peaks, in `mz_mz_std_ms1` to calculate mz statistics of features, and in `RT_edges` to define edges for baseline correction.
