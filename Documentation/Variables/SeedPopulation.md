## Description

SeedPopulation` is a NumPy array storing the initial set of parameters, used by the mating functions (`Mate_squareColumns_GaussParPop`, `Mate_square_WildPop`, `Mate_square_GaussParPop`) to generate new populations of parameters during optimization.
