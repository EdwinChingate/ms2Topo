## Description

SpectraList` is a list of NumPy arrays, where each array represents a mass spectrum. It's used in functions such as `CheckNeigbours`, `Similarity_AdjacencyMatrix`, `SimilarityNeigbours`, and `Retrieve_csv_ms2Spectra` to access and process individual spectra.
