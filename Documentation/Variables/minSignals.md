## Description

minSignals` is an integer that sets the minimum number of signals needed for a peak or feature to be considered valid, used in `ms2Peak`, `Find_ms2Peak`, `Chrom_ms1Peaks_Summaries`, `ms2_peak_stats`, `ms2_features_stats`, `mzPeak`, `Find_mzPeak`, `SplitChromatogram`, `ResolveFullChromatogram`, `ResolveChromPeaks`, `RetrieveChromatogram`, `AllSubChromatograms`, `PlotChromatogram`, `feat_ms2_Gauss`, `ModuleChromSummaries`, `ms2_peakStats_safe`, and `ms2_spectrum` to filter out less reliable data.
