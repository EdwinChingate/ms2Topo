## Description

TotalInt` is a numerical variable, used in `ms2Peak_contribution` to calculate relative peak contribution, in `WriteLog` for logging spectrum data, and in `ms2_peakStats_safe` for error logging, and it represents the sum of intensities in a mass spectrum.
