## Description

MS2_to_MS1_ratio` is an integer, used by `closest_ms1_spec`, `ms2_features_stats`, and `feat_ms2_Gauss` to define the maximum allowed difference between MS2 and MS1 spectrum IDs, for finding the closest related MS1 spectrum.
