## Description

Mutants` is a list of numpy arrays that store mutated parameters, used by `MutantOffspring`, `MutationTimes`, `Mate_MutantGenerations`, `MutateParameterLambda`, and `MutantansExtractor`, in the context of genetic algorithms to manage and use the mutated parameters of a population.
