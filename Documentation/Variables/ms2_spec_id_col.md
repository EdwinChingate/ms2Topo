## Description

ms2_spec_id_col` is an integer specifying the column index of the MS2 spectrum ID, used in `ms2_SpectralRedundancy`, `ms2_FeaturesDifferences`, `ms2_SpectralSimilarityClustering`, and `Retrieve_and_Join_ms2_for_feature` to correctly locate and retrieve the MS2 spectrum identifier.
