## Description

minSpec` is an integer defining the minimum number of spectra required for a chromatographic peak or feature to be valid, used in `ResolvingChromatogram`, `Chrom_ms1Peaks_Summaries`, `ReResolvingChromatograms`, `Feat_RT_edges`, `PlotChromatogram`, `ModuleChromSummaries`, and `ChromPeaksModules` to filter out less reliable features.
