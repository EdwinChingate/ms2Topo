## Description

MS2_features#ms2_FeaturesDF` refers to a pandas DataFrame, named `ms2_FeaturesDF`, generated by the function `Cluster_ms2_Features` based on the numpy array `MS2_features` output from the `feat_ms2_Gauss` function. This DataFrame stores the extracted MS2 features in a structured table.
