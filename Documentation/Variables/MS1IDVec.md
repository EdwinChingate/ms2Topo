## Description

MS1IDVec` is a numpy array storing MS1 identifiers and retention times, used across multiple functions (`Chrom_ms1Peaks_Summaries`, `closest_ms1_spec`, `ms2_features_stats`, `RetrieveChromatogram`, `PlotChromatogram`, `feat_ms2_Gauss`, `ModuleChromSummaries`, and `ExtractAllRawPeaks`) to link MS1 data to chromatographic and MS2 data.
