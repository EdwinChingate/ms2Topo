## Description

Modules` is a list of lists of integers that represents grouped features or signals, used by `ms2_feat_modules`, `Update_ids_FeatureModules`, `Summary_ms1_featModules`, `SignalsModulesStats`, `ms2_SpectralSimilarityClustering`, and `ChromPeaksModules` to process and summarize data related to those grouped features.
