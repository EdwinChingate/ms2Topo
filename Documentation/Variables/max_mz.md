## Description

max_mz` is a numerical value defining the upper limit of the mass-to-charge ratio (m/z). It is used in `ms2_SpectralRedundancy`, `mz_RT_spectrum_filter`, `AllMS2Data` and `ExtractAllRawPeaks` to filter spectra and extract data within the defined m/z range.
