## Description

mz_col` is an integer specifying the column index of the m/z value, used in `ms2_SpectralRedundancy`, `AdjacencyList_mz`, `AdjacencyListFeatures`, and `ms2_SpectralSimilarityClustering` to correctly locate and retrieve the m/z values.
